# Eskiwi React Native Application

This is the front-end application for Eskiwi, built using React Native and managed with Expo. This README provides instructions on how to run the project locally using Expo and how to build the app using EAS (Expo Application Services).

## Prerequisites

Before you begin, ensure you have the following installed on your local machine:

- [Node.js](https://nodejs.org/en/download/) (version 20.x or later)
- [npm](https://www.npmjs.com/get-npm) or [yarn](https://classic.yarnpkg.com/en/docs/install/)
- [Expo CLI](https://docs.expo.dev/get-started/installation/) (globally installed)
- [EAS CLI](https://docs.expo.dev/eas/cli/) (for building with EAS)
